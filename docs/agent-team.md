# 에이전트 팀 구성

Claude Learn Studio를 운영하는 5개 에이전트의 역할과 협력 구조입니다.

---

## 팀 구성

| 에이전트 | 역할 | 상태 |
|---------|------|------|
| 리서치 에이전트 | 최신 Claude 정보 수집·분류·요약 | 활성 |
| 커리큘럼 에이전트 | 학습 단계 구조화·난이도 설계 | 활성 |
| 실습 에이전트 | 코드 실행·디버깅·결과 검증 | 활성 |
| 평가 에이전트 | 진도 추적·맞춤 피드백 | 활성 |
| 멘토 에이전트 | Q&A·심화 가이드 | 활성 |

---

## 협력 파이프라인

```
주제 입력
    ↓
리서치 에이전트 → 최신 정보 수집 + 소스 검증
    ↓
커리큘럼 에이전트 → 단계별 구조화 + 난이도 설계
    ↓
실습 에이전트 → 실제 따라할 수 있는 실습 가이드 생성
    ↓
평가 에이전트 → 진도 체크 + 다음 단계 추천
    ↓
멘토 에이전트 → 질문 응답 + 심화 안내
```
