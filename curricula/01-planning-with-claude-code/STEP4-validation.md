# STEP 4 — 검증·보완: CPO·투자자 역할로 기획서 구멍 찾기

> **목표**: 외부 시선으로 기획서의 약점을 발견하고 보완  
> **소요 시간**: 40~50분

---

## 핵심 원칙

> **기획서의 치명적 약점은 본인이 못 찾습니다.**

Claude에게 CPO, 투자자, 비판적 동료 역할을 시켜서  
내 기획서의 구멍을 찾게 하는 방법입니다.  
— 브런치 Co-STAR 실전 가이드 참고

---

## 4-1. CPO 역할 검증 — 날카로운 질문 10개

### 프롬프트

```
당신은 시리즈B 스타트업의 CPO입니다.
아래 기획서를 읽고 전사 리뷰에서 실제로 할 법한
날카로운 질문 10개를 작성해주세요.

질문은 다음 4가지 관점을 반드시 포함해야 합니다:
1. 비즈니스 임팩트 (수익성, 시장 크기)
2. 고객 가치 (실제로 유저가 원하는가)
3. 기술 현실성 (실제로 만들 수 있는가)
4. 전략적 정합성 (회사 방향과 맞는가)

그런 다음, PO 입장에서 위 10개 질문 각각에 대해
구체적이고 논리적인 답변을 작성해주세요.
답변에는 실제 데이터, 가정, 우선순위 판단을 포함하세요.

[기획서 내용을 여기에 붙여넣기]
```

### 활용 방법

1. Claude가 제시한 질문 10개를 검토
2. **답변이 약한 질문 = 기획서에서 보강해야 할 섹션**
3. 약한 부분을 찾아 해당 섹션으로 돌아가 보완

---

## 4-2. 실패 케이스 검증

```
이 기획이 실패할 수 있는 이유 10가지를 제시해줘.

각 실패 이유에 대해:
- 발생 가능성 (상/중/하)
- 이미 기획서에 대응 방안이 있는지 여부
- 없다면 어떻게 보완할 수 있는지

[기획서 내용 붙여넣기]
```

---

## 4-3. 투자자 역할 검증

```
당신은 EdTech / AI 분야에 투자하는 시드 투자자입니다.
이 기획서를 읽고 투자를 결정하기 전에 반드시 확인하고 싶은
질문 5가지와, 현재 기획서에서 부족한 부분을 알려주세요.

[기획서 내용 붙여넣기]
```

---

## 4-4. 논리 흐름 & 설득력 최종 검토

```
이 기획서의 다음 4가지를 평가해줘:
1. 문장 구조 (읽기 쉬운가)
2. 논리 흐름 (앞 섹션이 다음 섹션을 자연스럽게 연결하는가)
3. 가독성 (단락 길이, 시각적 구분)
4. 설득력 ('왜 지금 해야 하는가'의 긴박감이 충분한가)

각 항목을 5점 만점으로 평가하고
개선이 필요한 부분을 구체적으로 알려줘.

[기획서 내용 붙여넣기]
```

---

## 4-5. 경쟁사 대응 시나리오 검증

```
경쟁사가 동일한 서비스를 출시할 경우
예상되는 대응 시나리오 3가지를 시뮬레이션해줘.
그리고 우리가 각 시나리오에서 어떻게 대응해야 하는지 전략도 제안해줘.
```

---

## 검증 결과 반영 방법

검증 후 약점을 발견했다면:

```
"방금 CPO 검증에서 [구체적 약점]이 발견됐어.
 [해당 섹션]을 이 피드백을 반영해서 다시 작성해줘."
```

---

### ✅ 체크리스트

- [ ] CPO 역할 검증 완료 (질문 10개 + 답변)
- [ ] 답변이 약했던 섹션을 보강했다
- [ ] 실패 케이스 10가지를 확인하고 대응 방안을 추가했다
- [ ] 논리 흐름 평가에서 4점 이상을 받았다
- [ ] 수정된 기획서를 파일로 다시 저장했다
